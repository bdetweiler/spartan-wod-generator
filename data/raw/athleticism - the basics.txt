The Basics

Running and burpees. Master these Spartan activities first and you will be #UNSTOPPABLE. It sounds simple, because it is. So no excuses. Get going.

Warm-up

    Dynamic warm up
    Run/cycle/row/jump rope 5 minutes

Main set

    Run 1 mile
    30 burpees

Repeat 1-4 times
